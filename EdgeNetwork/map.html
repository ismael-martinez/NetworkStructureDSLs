Steps for creating a set of candidate design edge locations.
  1. Enter the coordinates of the location where you want to design an edge infrastructure. Search.
  2. Select which type of device are being marked: IoT or Edge nodes.
  3. Click on the map to place markers for candidate edge locations.
  4. When you're done placing markers, click 'Export Coordinates" to export a JSON file of enumerated Lat/Lng coordinates.
